# forKatelyn
GDI program that draws the name "KATELYN"

![Alt Text](http://i.imgur.com/YuROLQk.gif)

## Building

Supported compilers: mingw-w64

Run the makefile to compile the GUI:

    mingw32-make.exe
## Libs used

WinAPI, GDI
